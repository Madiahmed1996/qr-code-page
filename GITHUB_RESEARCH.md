## Библиотеки и примеры для таблиц и табеля

* **TanStack Table (formerly React Table)** – библиотека с MIT‑лицензией, предоставляющая headless‑таблицы с возможностью редактирования. Используется для создания табеля и таблиц сотрудников/объектов. Примеров editable grid много, но мы собрали собственный компонент.
* Примеры готовых табелей на GitHub обычно закрыты или неполные, поэтому реализация табеля написана с нуля.

## Импорт и экспорт Excel

* **SheetJS/xlsx** (Apache‑2.0) – популярная библиотека для чтения и записи Excel‑файлов. Применяется как в браузере, так и на Node. Мы используем её для импорта и экспорта. Лицензия позволяет свободное использование при соблюдении условий.

## Выбор стека

На основании исследования выбран стек: React + TypeScript + Vite для фронтенда, Material UI для UI‑компонентов, TanStack Table для таблиц, Recharts для графиков, Express + better‑sqlite3 для backend. better‑sqlite3 имеет MIT‑лицензию【689636308851142†L0-L23】, обеспечивает высокую производительность и работу с реальным SQLite‑файлом. Такой подход не требует wasm и позволяет хранить данные локально. Desktop‑версия будет рассматриваться позднее на основе Tauri или Electron; при этом можно подключить `tauri-plugin-sqlite` или аналогичный плагин, как описано в README `tauri-plugin-sqlite`【11550961344158†L13-L27】. Пока что выбран Node/SQLite для упрощения CI.
