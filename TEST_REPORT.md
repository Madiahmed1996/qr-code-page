# Test Report

Этот файл будет автоматически обновлён после прохождения GitHub Actions.  CI‑workflow (`.github/workflows/ci.yml`) выполняет следующие проверки:

* `npm ci` — установка зависимостей.
* `npm run build` — сборка фронтенда в режим production.
* `npm test -- --run` — запуск модульных тестов на Vitest.
* `npx tsc --noEmit` — проверка типов TypeScript.
* `npm audit --audit-level=high` — проверка зависимостей на наличие известных уязвимостей.

После успешного завершения workflow в этом файле будет описан номер выполнения, выведенные команды и результаты тестов.
